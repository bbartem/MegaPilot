Назначение выводов HX711:
VCC, GND - питание
DT, SCK — информационные выводы
Е+, Е- — питание тензомоста
А+, А- — дифференциальный вход канала А
В+, В- — дифференциальный вход канала В

Библиотека предназначена для взвешивания груза за вычетом веса тары.
После включения устройства производится автоматическая калибровка весов на 0.
Для взвешивания необходимо установить пустую тару на весы, включить устройство, дождаться окончания калибровки и поместить груз в тару. 

Функция tare() сбрасывает весы на 0.
Согласно спецификации, когда пин SCK меняет значение с LOW на HIGHТ и остается в этом режиме дольше 60 сек., чип HX711 переходит в режим сниженного энергопотребления.
Функция power_down() переводит HX711 в режим сниженного энергопотребления.
Функция power_up() выводит устройство из режима сниженного энергопотребления.
Функция set_gain() без параметра включает высокий коэффициэнт усиления на канале A (при питании от 5V).
Функция read() возвращает необработанное значение, считанное с АЦП.
Функция read_average(20) возвращает среднее из 20-и считанных с АЦП значений.
Функция get_value() возвращает значение за вычетом веса тары.
Функция get_units() возвращает значение за вычетом веса тары, деленое на параметр SCALE, устанавливаемый функцией set_scale().
Функции get_value() и get_units() могут получать параметр times. В этом случае они возвратят среднее значение из полученного множества, вместо единожды считанного значения.


Калибровка устройства:
1. Вызовите функцию set_scale() без параметров.
2. Вызовите функцию tare() без параметров.
3. Поместите предмет вес которого Вам известен на весы и вызовите функцию get_units(10).
4. Разделите значение параметра полученного в шаге 3 на известный Вам вес и Вы получите значение параметра, которое необходимо установить в функции set_scale().
5. Скорректируйте значение параметра полученного в шаге 4, пока не получите точное значение.
